# screen

Get information about screen size, display, cursor position, etc.

## [hashtag](https://developer.eagle.cool/plugin-api/api/screen\#z1a5y)    Methods

## [hashtag](https://developer.eagle.cool/plugin-api/api/screen\#tkp0d)    getCursorScreenPoint()

Absolute position x, y of the current mouse cursor.

- Returns `Promise<point: Object>`



  - `point` Object



    - `point.x`

    - `point.y`


Copy

```
let point = await eagle.screen.getCursorScreenPoint();
```

* * *

## [hashtag](https://developer.eagle.cool/plugin-api/api/screen\#sskcn)    getPrimaryDisplay()

Returns primary display information.

- Returns `Promise<display: Display>`



  - `display` [Displayarrow-up-right](https://www.electronjs.org/docs/latest/api/structures/display) Object - Current display information.


Copy

```
let display = await eagle.screen.getPrimaryDisplay();
```

* * *

## [hashtag](https://developer.eagle.cool/plugin-api/api/screen\#eev58)    getAllDisplays()

Returns an array Display\[\], representing currently available screens.

- Returns `Promise<displays: Display[]>`



  - `displays` [Displayarrow-up-right](https://www.electronjs.org/docs/latest/api/structures/display)\[\]


Copy

```
let displays = await eagle.screen.getAllDisplays();
```

* * *

## [hashtag](https://developer.eagle.cool/plugin-api/api/screen\#ox9dk)    getDisplayNearestPoint(point)

Gets the plugin window coordinates x and y.

- `point` Object



  - `point.x` Integer type

  - `point.y` Integer type


- Returns `Promise<display: Display>`



  - `display` [Displayarrow-up-right](https://www.electronjs.org/docs/latest/api/structures/display) Object - Current display information.


Copy

```
let display = await eagle.screen.getDisplayNearestPoint({ x: 100, y: 100 });
```
