# drag

Implement native system file drag-and-drop functionality.

#### [hashtag](https://developer.eagle.cool/plugin-api/api/drag\#z1a5y)    Method

## [hashtag](https://developer.eagle.cool/plugin-api/api/drag\#tkp0d)    startDrag(filePaths)

Display notification window

- `filePaths` string\[\] - File paths to drag

- Returns `Promise<>`


Copy

```
await eagle.drag.startDrag([\
    "C:\\Users\\User\\Pictures\\drag1.jpg",\
    "C:\\Users\\User\\Pictures\\drag2.jpg",\
]);
```

circle-info

Note: This function is similar to Electron API's [webContents.startDrag(item)arrow-up-right](https://www.electronjs.org/en/docs/latest/tutorial/native-file-drag-drop) feature.
