# Глоссарий

| Термин | EN | Определение |
|--------|----|-------------|
| Токен | Design Token | Именованное значение дизайн-решения (цвет, размер, отступ) |
| Примитив | Primitive Token | Базовый токен с абсолютным значением (`blue.600 = #2563EB`) |
| Семантический | Semantic Token | Токен-алиас с ролью (`color.action.primary → {blue.600}`) |
| Компонентный | Component Token | Токен для конкретного компонента (`button.bg.default`) |
| Тема | Theme | Набор переопределений токенов для бренда |
| DTCG | Design Tokens Community Group | W3C стандарт формата токенов |
| MOC | Map of Content | Навигационная заметка со ссылками на связанные заметки |
| ADR | Architecture Decision Record | Запись архитектурного решения |
| Паттерн | Pattern | Повторяемое UX-решение типовой задачи |
| Эвристика | Heuristic | Принцип / правило для оценки юзабилити |
| HIG | Human Interface Guidelines | Гайдлайны Apple для iOS |
| MD3 | Material Design 3 | Гайдлайны Google для Android |
| Freshness | Актуальность | Статус свежести контента заметки |
| Agentic UX | — | UX для AI-агентов с автономным поведением |
| HITL | Human-in-the-Loop | Человек контролирует / одобряет AI-действия |
