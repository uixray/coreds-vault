# Как внести вклад

## Для пользователей vault

1. Найдите ошибку или устаревшую информацию
2. Создайте Issue на GitHub
3. Или создайте Pull Request с исправлением

## Для контрибьюторов

1. Fork репозитория
2. Создайте ветку: `feature/add-pattern-name` или `fix/update-token-name`
3. Следуйте [[conventions|конвенциям оформления]]
4. Убедитесь что frontmatter заполнен корректно
5. Создайте PR с описанием изменений

## Что можно добавлять

- Новые паттерны в `02-patterns/`
- Исправления и дополнения к существующим паттернам
- Примеры из реальных продуктов
- Переводы на другие языки
- Скрипты автоматизации

## Чего не добавлять

- Проприетарный контент (скриншоты под NDA)
- Рекламу инструментов
- Контент без frontmatter
